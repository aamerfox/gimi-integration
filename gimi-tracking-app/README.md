# SaudiEx Tracking App

A modern, high-performance web application for real-time IoT tracking, history playback, and geofencing. Powered by the **TrackSolid Pro IoT API** for reliable, real-time telemetry.

## ✨ Features
-   **Live Tracking**: Real-time position updates and device status.
-   **History**: Full movement playback with data-rich charts.
-   **Geofencing**: Create and manage virtual boundaries with instant alerts.
-   **Share Link**: Generate temporary, secure tracking links for sharing.
-   **Multi-Language**: Full support for English and Arabic (RTL).

## 🚀 Getting Started

### Prerequisites
-   Node.js (v18+)
-   NPM or Yarn

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

## 🧪 Testing
The project includes a comprehensive Playwright E2E suite covering over 57 test cases.
```bash
npm test
```
